Project Overview
This is a simple Android application built to help people stay connected. It suggests small, friendly ways to reach out to friends and family based on the time of day.

## Main Features
Smart Suggestions: Get ideas for messages based on the time you enter (like "Morning" or "Night").

Simple Design: A clean, easy-to-use interface with a calm blue background.

One-Tap Reset: Quickly clear your input and start again with a single button.

## How to Use the App
Enter Time: Type the current time of day in the input box.

Get Idea: Tap the main button to see a social suggestion.

Connect: Use the suggestion to send a text or call someone.

Clear: Use the reset button to refresh the screen.

## Technical Details
Language: Kotlin

Platform: Android

Tools Used: Android Studio and XML for the layout.

Core Logic: The app uses simple "if-else" logic to match the time of day with the best social tip.

## Setup Instructions
Download the project files.

Open the project in Android Studio.

Click the Run button to install it on your device or emulator.
